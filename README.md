# Melffy 丈感ガイド

## プロジェクトについて

Melffyのお客様が、身長と商品の長さを入力するだけで、
ワンピース・スカート・パンツの丈感を視覚的に確認できるWebサービスを開発しています。

---

## コンセプト

「可愛いを後回しにしない」

サイズで迷って購入を諦めるのではなく、
安心してお買い物を楽しめる体験を目指します。

---

## Ver.1で実装する機能

* 身長入力
* アイテム選択（ワンピース・スカート・パンツ）
* 商品の長さ入力
* モデルで丈感を表示
* 丈感コメント表示
* スマホ専用デザイン
